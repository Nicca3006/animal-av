Welcome to Exercise Editor wizard!

Using this front-end, you can easily create a new exercise or edit an existing one. This process is divided in 5 steps. Each step is represented by a special view.

In order to navigate through the views, use the buttons at the bottom of the window.

Following views are available:

  1. Algorithm Choose an a algorithm for the exercise.
  2. Parameter Define the initial arguments of the algorithm.
  3. Description Set a title, subtitle and description.
  4. Questions Manage the list of questions. Each question is edited in a separate editor front-end.
  5. Finish Persist the exercise.

Press Next to start.
