Suchalgorithmen erhalten im Allgemeinen ein Problem als Eingabe
und suchen nach einer geeigneten Lösung. In den hier
abgedeckten einfachen Füllen ist das Problem das
Auffinden eines bestimmten Elements in einer Menge (zumeist
einem Array oder einer Zeichenfolge).

Falls bekannt ist, dass die Suchmenge sortiert ist, können
sehr effiziente Algorithmen eingesetzt werden.
